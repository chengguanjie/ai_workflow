# 基础工艺与口感控制（Demo 知识库）

本文件为模拟知识库文档，用于 RAG 检索结果“真实被调用”的验证。

## 蔬菜处理（保持脆嫩）

- 焯水 3–5 分钟：颜色更鲜亮、口感更脆
- 焯水后立刻过冷水可进一步保持口感（家用可选）

## 蛋白主料处理

### 鸡胸肉（常见问题：柴）

- 切片/切条增大受热面积，缩短加热时间
- 腌制时加入少量酸（柠檬汁/米醋）可改善口感（注意不要过量）

### 豆腐（常见问题：出水、易碎）

- 压水 5–10 分钟减少出水
- 中小火煎至表面金黄再翻面，定型后更不易碎

## 酱汁（少油也好吃）

- 酸味（醋/柠檬）提升“清爽感”
- 蒜末、黑胡椒提升“香味密度”
- 盐度控制：先少后多
