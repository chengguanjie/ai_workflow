# 输出格式规范（Demo 知识库）

用于演示：工作流在生成配方时，如何输出结构化 JSON，并在后续节点中做展示优化。

## 推荐 JSON 字段

- recipeName: string
- servings: number
- cuisine: string
- constraints: object（如 highProtein/lowSugar/lowFat/vegan/glutenFree）
- ingredients: array（name/amount/unit/note）
- steps: array（string）
- nutritionEstimate: object（perServing + note）
- rationale: object（summary + knowledgeSources）
- qualityChecklist: array（string）

## 引用来源（可追溯）

在 rationale.knowledgeSources 中输出被引用的“文档名列表”，用于验证检索是否发生以及是否被使用。
