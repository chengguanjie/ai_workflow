# 过敏原与替代方案（Demo 知识库）

本文件用于演示“知识库文档建立 → 工作流引用 → 执行检索调用”的全链路验证（内容为模拟）。

## 常见过敏原

- 花生/坚果：可用南瓜籽、葵花籽替代点缀；或直接省略
- 乳制品：可用椰浆、燕麦奶、豆奶替代（注意风味变化）
- 麸质（小麦/酱油）：可用无麸质酱油（tamari）或椰子氨基酸（coconut aminos）
- 大豆：对大豆过敏时避免豆腐、豆干、酱油；主蛋白可改鸡胸/鱼/蛋

## “无麸质”处理提示

- 重点是避免普通酱油、麦制品、面包糠
- 选择“无麸质酱油/豉油”并在配方中明确标注

## “纯素/素食”处理提示

- 纯素：不含肉、蛋、奶、蜂蜜
- 高蛋白纯素：豆腐/豆干/毛豆/鹰嘴豆等组合，避免单一来源导致口感疲劳
